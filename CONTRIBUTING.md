# Contributing to NG

Thank you for your interest in contributing.

NG is a proprietary language and compiler project. Contributions are accepted under strict terms.

## By contributing, you agree to the following:

- All contributions become the exclusive property of the project owner.
- You do not retain any rights over your contribution.
- You grant the author full permission to use, modify, and relicense your contribution without limitation.
- You will not reuse or redistribute the contributed code outside this repository.

## What you get:

- Your name may be listed in contributors (at the author's discretion)
- Experience working on a language and compiler project

## Contribution Guidelines:

- Follow the existing architecture and project direction
- Do not submit copied or third-party licensed code
- Keep contributions focused and well-documented

## How to contribute:

1. Fork the repository
2. Create a feature branch
3. Submit a pull request with a clear description

## Important

If you do not agree with these terms, do not contribute.
